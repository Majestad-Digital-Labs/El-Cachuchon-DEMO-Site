EL CACHUCHON — Poster Layout (Centered)

How to open:
1) Unzip the folder.
2) Double-click index.html
   OR on Mac terminal:
   open index.html

Recommended (best):
- Install the VS Code extension “Live Server”
- Right-click index.html -> “Open with Live Server”

Files:
- assets/img/background.jpg  (your full “poster” background)
- index.html                (home: 6 hats centered in the black middle)
- hats.html                 (view all)
- wholesale.html            (wholesale info)
- contact.html              (simple contact/order form demo)

Quick tuning:
- Open assets/css/styles.css
- Adjust these variables:
  --poster-max   (overall “paper size” width)
  --overlay-top  (moves the hat display up/down inside the black middle)
  --overlay-width (makes the center display wider/narrower)
